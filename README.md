# POS

## 프로젝트 소개

삼촌께서 식당을 개점 하셨는데 아직 포스기 소프트웨어가 적용이 안되어있어 수기로 작성하는것을 보고 필요한 기능만 구현하여 사용하게 만들어드림.

## 시스템 기능

- 테이블 수 조절
- 상품 추가
- 판매 기능
- 판매 내역
